
1️. var, let, const?

answer:
var → old variable, function scope

let → changeable variable

const → cannot change value


2️. Spread Operator ...?
answer:
Used to copy or expand arrays/objects


3️. map() vs filter() vs forEach()?

answer:

map() → change items → returns new array

filter() → select items → returns new array

forEach() → loop only → no return


4️. Arrow Function?

answer:

Short way to write a function.


5️. Template Literals?

answer:

String with variables inside using backticks.
